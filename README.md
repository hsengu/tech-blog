## Installation
    =============
    = Git Setup =
    =============
	$ git clone https://github.com/hsengu/tech-blog
	$ cd ./tech-blog
	$ npm install

    ==================
    = MySql DB Setup =
    ==================
    $ mysql -u your_username -p
    source db/schema.sql
    quit
    
    ==============================
    = Seed DB with Sequelize ORM =
    ==============================
    $ npm run seed
